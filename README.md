# thingy-server

## Getting started

### Dependencies
- `docker`
- `docker-compose`

### How to run

- `git clone git@github.com:SnipyJulmy/hesso-latextemplate-pa.git`
- `docker-compose run -d --build`
- server is now listening on http://localhost:8080
- datas from mysql and influxdb are stored into `./data`

### How to stop
- `docker-compose down`
